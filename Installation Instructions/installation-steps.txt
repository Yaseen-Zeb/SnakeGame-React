Installation Steps
================

1. Download zip file and Unzip file it.
2. cut "Api React" folder from downloaded folder/src/php/php/"Api React" and paste inside htdocs folder.

Database Configuration:
==================
1. Open phpmyadmin
2. Create New Database named "react_crud".
3. Import database tables in newly created database "react_crud" from downloaded folder -> sql -> users.sql.
5. Open Your browser put inside URL: "http://localhost/3000/"

At Last :
Run react app in installed folder with command "npm start" in promt command.
